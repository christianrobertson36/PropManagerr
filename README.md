# PropManagerr - Under Development

UK Landlord Property Management Platform

PropManagerr is a self-hosted property management application for UK landlords, providing:

- Property Management
- Tenant Management
- Rent Tracking
- Maintenance Tickets
- Compliance Documents
- Expense Tracking
- Dashboard Reporting

## Features

### Authentication
- Admin login
- Tenant login
- JWT authentication

### Properties
- Add properties
- Edit properties
- Delete properties

### Tenants
- Add tenants
- Edit tenants
- Delete tenants
- Property assignment

### Rent Management
- Track rent payments
- Edit payments
- Payment status tracking

### Compliance Documents
- Document management
- Expiry tracking
- Compliance reminders
- File upload support

### Maintenance
- Tenant repair requests
- Ticket tracking
- Status management
